Exercício 1.5

Escreva uma função radianos(graus,mins,segs) que, dado o valor de um ângulo em graus, minutos e segundos, converte-o para radianos. Relembre que \(360º\) corresponde a \(2 \pi\) radianos, cada grau tem 60 minutos e cada minuto tem 60 segundos.

def radianos(graus,mins,segs):
    mins = mins + segs / 60
    graus = graus + mins / 60
    return graus * (2 * math.pi / 360)
print(radianos(0,0,60))
print(radianos(0,60,0))
print(radianos(180,10,40))

Defina uma função forte(passwd) que verifica se uma palavra-passe, dada pela cadeia de caracteres passwd, é forte. Considera-se que a palavra-passe é forte se tiver 8 caracteres ou mais, e incluir pelo menos uma letra maiúscula, uma letra minúscula e um algarismo. O resultado deve ser um valor lógico (True ou False).

>>> forte('9EwL56')
False
>>> forte('HXKW1393')
False
>>> forte('ffu4G7Fghjk')
True

def forte(passwd): maiusculas = { c for c in passwd if c.isupper() } minusculas = { c for c in passwd if c.islower() } algarismos = { c for c in passwd if c.isdigit() } return len(passwd) >= 8 and len(maiusculas) > 0 and len(minusculas) > 0 and len(algarismos) > 0

print(forte(‘9EwL56’)) print(forte(‘HXKW1393’)) print(forte(‘ffu4G7Fghjk’))
